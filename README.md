# Handbook

## DIY Mobile Firmware RM625

This firmware is based on the Nokia 5233 model. The main version of this firmware is the official V51.1.002 version. 

The conditions for flashing this time are:
- The machine must be the Nokia 5233 model
- The machine version must be less than or equal to V51.1.002 version (otherwise the program will crash)
- The method to check the phone version is to enter `*#0000#` on the standby state of the phone.

Now, let's introduce the usage of this firmware:

## Driver Installation

First, you need several software to write into the phone, they are:
- JAF flashing tool
- NOKIA PC Suite (of course you also need the corresponding mobile phone and data cable)

After you meet the above conditions, we start to write the firmware into the phone. The steps are as follows:
1. First install NOKIA PC Suite on the computer (after installation, you need to turn off all security guards and antivirus software on the computer, because it may block the driver and cause installation failure)
2. After installing PC Suite, connect the phone to the computer. Choose the first option--PC Suite/OVI when the phone pops up. Then the computer will find new hardware. Update the hardware when the update dialog box pops out. After updating successfully, right click on the "My Computer" icon and select "Device Manager". The interface should look like the following:

![image](https://github.com/AndyFCui/NOKIA_5233_firmware/assets/10085168/036298c3-edfe-4280-af4f-4b7b3eb31608)
![image](https://github.com/AndyFCui/NOKIA_5233_firmware/assets/10085168/df440234-6662-4066-a1e9-e0cebcc78af2)

Next, install JAF flashing tool on the computer. After the installation, there will be an icon that looks like a football.

![image](https://github.com/AndyFCui/NOKIA_5233_firmware/assets/10085168/fa17c994-c7b0-4ad4-8e5f-e12875099556)

Then unzip my firmware. Don't change the path!!!!!! I have made it a self-extracting file that can be run directly!

## JAF File Guide

Now everything is ready (JAF + firmware + computer + phone + connection data cable). Let's start the flashing process with JAF software. Follow the 1.2.3.4.5 steps in the picture below!

![image](https://github.com/AndyFCui/NOKIA_5233_firmware/assets/10085168/24cfb197-08ec-4034-943f-8fa8ad7e668b)

Click on step 1, tick the two options in step 2, cancel the one in step 3, tick the one in step 4, and the dialog box in step 5 will pop out. Find the chip type of the phone: `5230 RM-588` `5233 RM-625` Then confirm! The following picture will appear:

![image](https://github.com/AndyFCui/NOKIA_5233_firmware/assets/10085168/d6b17307-d088-4737-a407-10d365a64eea)

After clicking "Confirm", go back to the interface and click the `FLASH` button. The following picture will appear:

![image](https://github.com/AndyFCui/NOKIA_5233_firmware/assets/10085168/4c1b46c1-760a-4dd8-a532-489e2d9693e5)

When "Yes" "No" appears, don't click "Yes" yet. First turn off the phone, then click "Yes". Then, the following pictures will appear:

![image](https://github.com/AndyFCui/NOKIA_5233_firmware/assets/10085168/c2f9b2f9-797a-4135-b7b3-e7f6093e1a2b)
![image](https://github.com/AndyFCui/NOKIA_5233_firmware/assets/10085168/540c369c-2156-47c1-9893-09c35e21044b)

After clicking "Yes", press the power button of the phone for 1 to 2 seconds. Once the progress bar below moves, let go of the power button. The green progress bar will start moving. Wait until it reaches 100%. At this point, the flashing process is complete.

![image](https://github.com/AndyFCui/NOKIA_5233_firmware/assets/10085168/9e10aeb4-082c-4fb5-b216-05a115af7c4c)
![image](https://github.com/AndyFCui/NOKIA_5233_firmware/assets/10085168/d3cfed23-06fd-4136-b7cb-5d8e67377b6e)

If after pressing the power button for 2 seconds, you hear a sound from the computer but the progress bar does not move, please unplug the data cable directly, then start the phone and connect it to the computer, select the first option for connection, then click the `FLASH` button, then turn off the phone, click "Yes", and then press the power button for 2 seconds to start flashing.

![image](https://github.com/AndyFCui/NOKIA_5233_firmware/assets/10085168/60ee12e7-908c-4f1f-aee5-3a19011923d5)

After flashing, my firmware can be used. As I explained at the beginning, it is modified from the official V51.1.002 version of the original firmware.

## DIY Mobile Firmware RM625

DIY手机固件RM625这款基于诺基亚5233的手机固件，这款固件的主版本为官方的V51.1.002版本，
此次刷机的条件为：
- ★机器必须为诺基亚5233机型
- ★	机器版本必须小于或等于V51.1.002版本（否则会程序崩溃）
- ★	看手机版本的方法是在手机待机状态下输入 `·*#0000#`

下面为各位老师介绍下这款固件的使用方法：

`驱动安装`
- 1. 首先它需要几款软件来执行写入到手机中，他们分别是：
    - JAF强刷工具
    - NOKIA PC套件（当然还需要对应的手机和数据线）

- 以上条件具备以后我们开始操作把固件写入到手机中，方法如下：
1. 首先安装NOKIA PC套件到电脑上，（安装后需要关闭计算机上所有安全卫士和杀毒软件，因为可能会被屏蔽驱动导致安装失败）
